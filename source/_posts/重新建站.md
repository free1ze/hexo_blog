# hexo使用

[hexo官网](https://hexo.io/zh-cn/)
## 新建文件

``` bash
$ hexo new "My New Post"
```

## 本地测试

``` bash
$ hexo server
```

## 生成并部署

``` bash
$ hexo clean
$ hexo generate
$ hexo deploy
```

或使用简写

``` bash
$ hexo clean
$ hexo g
$ hexo d
```
